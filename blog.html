The first time I ever create a scary story in HTML.

HTML Homework - Revival Floor

Revival Floor is a scary/horror/creepy pasta type of a story genre. It was written in HTML present the story using text and images. I use about 15 differnt HTML files for the story because I want the user to navigate between different HTML files.

I implemented a custom CSS files to style the story. I made the image either small or move it around the page. I imported custom text font to make the atmosphere scary. Added sounds later to up the creepy factor. Although it was a story, it was more of an interactive story where the user can interact with the page. I had implement some JavaScripts to make all the HTML more interactive.

I made it so that you can click buttons to change images, other HTML pages are locked until user posses a certain item, and keeping track of items the user will find throughout the story.
